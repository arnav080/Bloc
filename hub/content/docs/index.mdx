Welcome to the **Bloc Hub** documentation. Bloc is a local-first model registry and deployment toolchain for running optimized, reproducible local AI environments in minutes.

## Core Features

- **1-Command Deployments**: Pull and run fully optimized local LLMs instantly.
- **100% Local**: No API tokens, no telemetry, no leaks. Everything stays on your metal.
- **Portability**: Write a single configuration blueprint and share identical environments across your team.

## Quick Start

To deploy and run your first model, use the Bloc CLI in your terminal:

```bash
bloc deploy arnav/qwen-3.5-9b-super
```

Explore our guides to configure your GPU acceleration, custom quantizations, and system APIs!
