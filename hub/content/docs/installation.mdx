title: Installation
description: Install the Bloc CLI on macOS, Linux, and Windows.
---

Bloc is distributed as a single lightweight, statically compiled binary with zero external dependencies. Choose the installation method for your platform below.

---

## 🍎 macOS

### Method 1: Via Homebrew (Recommended)
Add the official tap and install `bloc`:
```bash
brew tap arnav080/bloc
brew install bloc
```

### Method 2: Manual Tarball Installation
Download and extract the binary matching your hardware, then move it into your system PATH:

#### Apple Silicon (M-Series)
```bash
curl -LO https://github.com/arnav080/Bloc/releases/download/v0.1.0/bloc_darwin_arm64.tar.gz
tar -xzf bloc_darwin_arm64.tar.gz
sudo mv bloc /usr/local/bin/
```

#### Intel Core
```bash
curl -LO https://github.com/arnav080/Bloc/releases/download/v0.1.0/bloc_darwin_amd64.tar.gz
tar -xzf bloc_darwin_amd64.tar.gz
sudo mv bloc /usr/local/bin/
```

## 🐧 Linux

### Method 1: Debian / Ubuntu (AMD64 `.deb`)
```bash
wget https://github.com/arnav080/Bloc/releases/download/v0.1.0/bloc_0.1.0_linux_amd64.deb
sudo dpkg -i bloc_0.1.0_linux_amd64.deb
```

### Method 2: RedHat / Fedora (AMD64 `.rpm`)
```bash
sudo dnf install https://github.com/arnav080/Bloc/releases/download/v0.1.0/bloc_0.1.0_linux_amd64.rpm
```

### Method 3: Manual Tarball Installation (Any Distro)
```bash
curl -LO https://github.com/arnav080/Bloc/releases/download/v0.1.0/bloc_linux_amd64.tar.gz
tar -xzf bloc_linux_amd64.tar.gz
sudo mv bloc /usr/local/bin/
```

## 🪟 Windows

### Method 1: Windows Installer (Recommended)
1. Download the **`bloc_windows_amd64.msi`** installer from the [Releases page](https://github.com/arnav080/Bloc/releases/tag/v0.1.0).
2. Double-click the installer and follow the setup prompt.
3. Open a **new** command prompt (`cmd`) or PowerShell terminal to refresh your environment variables, and run `bloc` globally.

### Method 2: Via PowerShell (Manual Zip Setup)
Run the following commands to download, extract, and append the folder to your User PATH:
```powershell
Invoke-WebRequest -Uri "https://github.com/arnav080/Bloc/releases/download/v0.1.0/bloc_windows_amd64.zip" -OutFile "bloc.zip"
Expand-Archive "bloc.zip" -DestinationPath "$env:USERPROFILE\bin" -Force
[Environment]::SetEnvironmentVariable("Path", $env:Path + ";$env:USERPROFILE\bin", "User")
```

---

## Post-Installation Verification

Confirm your installation was successful and is accessible in your current shell:

```bash
bloc --help
```
This should output the version info, usage help, and a list of available subcommands. If you see a `command not found` error, verify that the folder containing `bloc` is included in your system’s `PATH` variable.
